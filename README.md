# Titanic Survival Prediction 🚢

## 📌 Project Overview
This project analyzes the Titanic dataset and predicts whether a passenger survived or not using Machine Learning.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## ⚙️ Features Used
- Age
- Sex
- Pclass
- Fare
- FamilySize

## 🤖 Model Used
Logistic Regression

## 📊 Results
The model achieved an accuracy of around 80%. The confusion matrix shows that most predictions were correct.
